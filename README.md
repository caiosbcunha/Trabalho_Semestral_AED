# Caminho do Conhecimento

Jogo de tabuleiro educativo em C para 2 a 4 jogadores, desenvolvido 
como projeto da disciplina de Algoritmos e Estruturas de Dados do 
SENAI CIMATEC. Os jogadores percorrem 45 casas respondendo perguntas 
sobre programação em C e estruturas de dados, com casas de Sorte, 
Revés e Pergunta. O projeto aplica na prática Fila (controle de 
turnos), Pilha (banco de perguntas por unidade temática) e Lista 
Duplamente Encadeada (tabuleiro navegável).
